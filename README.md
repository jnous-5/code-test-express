# HTHA Developer - Coding Exercise

## Getting Started

### Prerequisites

Before getting started, ensure you have the following installed on your machine:

- [Docker](https://www.docker.com/get-started)
- [VSCode](https://code.visualstudio.com/) with the [Dev Containers extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)

### Installation

Clone the repository

```sh
git clone https://github.com/jnous-5/code-test-express.git
```

### Open in VS Code and click `Reopen in Container` when prompted

```sh
cd code-test-express
code .
```

### Development

```sh
# Install dependencies
yarn install

# Start development server
yarn dev
```

Open `http://localhost:4000/graphql` in your browser to see the app.
